# Clearning

## Prerequies

To have install make and cmake on your desktop, with ubuntu use :

```sh
sudo apt install make cmake
```

## Compilation

Go to the root path of the project and execute :

```sh
mkdir build
cd build
cmake ../CmakeList.txt
make
```

## Main Execution

```sh
./puissance4
```

## Test Execution

```sh
ctest or ./tests
```
